COURSEWORK

EECS 281 - Data Structures and Algorithms in C++

An introduction to algorithm analysis and O-notations. This course emphasizes fundamental data structures including lists, stacks, priority queues, hash tables, trees (binary, search and balanced), and graphs. This course also covers searching and sorting algorithms, recursive and basic graph algorithms, and requires students implement greedy algorithms and divide and conquer strategy. This course requires 4 major programming assigments, along with regular labs and two major exams.

EECS 445 - Introduction to Machine Learning

Theory and implementation of state-of-the-art machine learning algorithms for large-scale, real-world applications. Topics include supervised learning (regression, classification, kernel methods, neural networks, and regularization) and unsupervised learning (clustering, density estimation, and dimensionality reduction). For each topic, mathematical principles, key algorithmic ideas, and implementation are highlighted.

SI 422 - Needs Assessment and Usability Evaluation

This course, requires that students evaluate interactive software products, focusing on social computing services, through hands-on experience with a range of methods and design issues. These methods include competitive analysis, in-depth evaluation of user needs using observation and ethnographic methods. Students also conduct user research through interviews and surveys, constructing personas and scenarios based on target users while using heuristics to evaluate systems from these perspectives (heuristic evaluation, cognitive walkthroughs, card sorting). This course also requires comprehensive usability testing, with comprehensive methods of improving user interface details and design considerations, and the monitoring and testing of system designs through experiments and log analysis. This course requires students to create detailed, development-team-ready product reports, and to be able to conduct proper, thorough usability testing.

MKT 313 - Consumer Behavior

The main objective of this course is to formulate appropriate and effective marketing strategies with a clean and accurate understanding of consumers. The objectives of the course are to introduce the student to concepts developed in psychology, economics, and sociology and their relationship to consumer behavior, while involving the student directly in the study and analysis of consumer behavior. Students will develop the ability to translate what can be learned into marketing action implications.

EECS 370 - Introduction to Computer Organization

Basic concepts of computer organization and hardware. Instructions executed by a processor and how to use these instructions in simple assembly-language programs. Stored-program concept. Data-path and control for multiple implementations of a processor. Performance evaluation, pipelining, caches, virtual memory, input/output.
